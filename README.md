## **Overview**
This application provides an interactive way to search for Pok√©mon, view detailed profiles, and explore their evolutions. It features an autocomplete search bar, clickable profiles, and intuitive navigation, making it a seamless experience for Pok√©mon fans.

---

## **Features**

### **Search Bar with Autocomplete**
- Begin typing a Pok√©mon's name in the search bar to see a list of suggestions.
- Helps quickly locate Pok√©mon without needing the exact spelling.

### **Pok√©mon Profile**
- Each profile includes:
  - **Image and Name**
  - **Navigation Bar** for Info, Skills, and Moves:
    - **Info:** Displays general information.
    - **Skills:** Lists the Pok√©mon's abilities.
    - **Moves:** Shows moves and evolution details.
- Clicking on any evolution will navigate to that Pok√©mon's profile.

### **Navigation**
- The app logo and home icon in the header return users to the home page.
- Profiles include a search bar to quickly navigate to other Pok√©mon profiles.

---

## **How to Use**

1. **Starting the Search**:
   - Enter the name of the Pok√©mon into the search bar on the home page.
   - Use the autocomplete feature for suggestions.

2. **Viewing a Pok√©mon's Profile**:
   - Select a Pok√©mon from the autocomplete suggestions or hit the "Search" button to load its profile page.

3. **Exploring the Profile**:
   - Use the navigation buttons (Info, Skills, Moves) to explore different sections of the profile.

4. **Exploring Evolutions**:
   - On the Moves page, view the Pok√©mon's evolutions.
   - Click on any evolution to navigate to that Pok√©mon's detailed profile.

5. **Navigating the App**:
   - Use the app logo or the home icon to return to the home page.
   - Use the search bar within profiles to navigate to other Pok√©mon profiles.
